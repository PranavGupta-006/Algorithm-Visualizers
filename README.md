# Algorithm Visualizers

Algorithm Visualizers is a collection of interactive tools designed to help students understand how algorithms work internally. Instead of only reading code or theoretical explanations, users can observe algorithms step by step through visual representations.

The goal of this project is to make learning algorithms more intuitive and engaging by turning abstract computational processes into visual simulations.

---

# Motivation

Algorithms are often difficult to understand when studied only through pseudocode or textbooks. Visualization allows learners to see how data changes during execution, making it easier to grasp concepts such as iteration, recursion, and complexity.

This project was built as a learning tool for students studying computer science, particularly in courses covering data structures and algorithms.

---

# How It Works

Each algorithm is implemented alongside a visualization layer that illustrates its internal operations.

Typical visualizations show:

* comparisons between elements
* swaps or movements in data structures
* traversal of nodes in graphs
* recursive calls and intermediate states

This allows users to follow the algorithm’s logic in real time.

---


The project is organized so that algorithm logic and visualization logic remain separate, making it easier to maintain and extend.

---


# Usage

1. Use the Streamlit Website
2. Choose an algorithm from the available list.
3. Provide input data if required.
4. Observe the algorithm execute step by step through the visual interface.

This makes it easier to analyze how the algorithm processes the data and understand its behavior.
